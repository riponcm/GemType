Gemini API key

Your key is stored only on this device (chrome.storage.local) and sent only to generativelanguage.googleapis.com.
  1. Open aistudio.google.com/apikey and sign in with a Google account.
  2. Accept the terms (first time only).
  3. Click Create API key — no credit card needed for the free tier.
  4. Copy the key (starts with AIza) and paste it above.
